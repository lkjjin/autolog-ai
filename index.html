AI 기반 로그 요약 및 작업 기록 도구

앱 소개

AutoLog.AI는 PC에서 오피스(Word, Excel, PowerPoint)와 메모장 저장 이벤트를 실시간으로 감지하여, Gemini 온라인 AI 또는 로컬 AI(Ollama 설치 모델)로 내용을 자동 요약합니다. 요약된 결과는 앱 내부에 저장되며, Notion으로도 자동 전송할 수 있습니다.

  • 실시간 자동 감지: 문서 저장 시 즉시 기록
  • AI 요약: Gemini(온라인) 또는 Ollama 모델(로컬)
  • 지원 로컬 모델: llama3, gemma2:2b, qwen2.5:1.5b
  • 자동 저장 및 Notion 전송
  • 작업 일지 자동 기록

다운로드

설치 안내

  1. 온라인 AI 사용 시: Gemini API 키를 설정창에 입력하세요.
  2. 로컬 AI 사용 시: Ollama를 설치하세요.
  3. CMD에서 원하는 모델을 설치합니다: 
    ollama pull llama3
ollama pull gemma2:2b
ollama pull qwen2.5:1.5b
  4. AutoLog.AI 실행 후 설정창에서 온라인/로컬 AI를 선택하세요.
